## 已知限制

### 当前版本（v0.2.0）
1. **仅支持3自由度**：XY平移 + Yaw旋转
   - Z轴高度锁定
   - Pitch/Roll姿态未使用
   - 适用于平地运动

2. **舵轮协调已禁用**：避免轮速抑制问题
   - 舵角未到位时轮速不会降低
   - 可能导致打滑（待优化）

3. **无轨迹记录**：当前版本不记录轨迹数据
   - 无法回放
   - 无法导出

### 下一步改进
- [ ] 重新启用并优化舵轮协调逻辑
- [ ] 实现6自由度支持（Z/Pitch/Roll）
- [ ] 添加轨迹记录与可视化
- [ ] 实时状态HUD显示

## 故障排查

### 问题1：VIO设备未连接
```
[VIO] 设备未连接
错误: VIO设备初始化失败
```
**解决**：
- 检查USB连接
- 确认设备权限：`sudo chmod 666 /dev/ttyUSB*`
- 运行VIO SDK示例验证设备

### 问题2：找不到VIO库
```
error while loading shared libraries: libcarina_vio.so
```
**解决**：
```bash
export LD_LIBRARY_PATH=/home/mxz/桌面/Ubuntu20.04/x86_64/lib:$LD_LIBRARY_PATH
```

### 问题3：底盘不动
**检查**：
1. VIO数据是否更新？（`--verbose` 查看）
2. 速度指令是否为0？
3. VIO设备是否在移动？

### 问题4：底盘乱动
**检查**：
1. VIO定位是否稳定？（观察位置跳变）
2. 是否有遮挡或光线不足？
3. 数据验证是否启用？
